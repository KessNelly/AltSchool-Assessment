# Student Grade Tracker

A simple JavaScript web application for tracking student grades.

## Features

- Add students with name and grade
- Display students dynamically using the DOM
- Calculate and update average grade automatically
- Delete students from the list
- Input validation for name and grade
- Highlights students scoring above average
- Data persistence using localStorage

## Technologies Used

- HTML
- CSS
- JavaScript
